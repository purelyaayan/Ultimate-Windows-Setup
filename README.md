# ✨ My Windows Rice Configs

Welcome to my personal desktop ricing setup repository!  
This repo contains all the configs and customization files I use to make my Windows desktop look clean, minimal, and aesthetic.

## 📦 Included Configs

### 🪟 GlazeWM
Custom window manager configuration for tiling windows and workflow improvements.

### 🎨 YASB
My custom YASB taskbar setup with widgets, styling, and layout configs.

### ⚡ Windhawk
A collection of my Windhawk mod settings and tweaks used for customizing Windows behavior and UI.
